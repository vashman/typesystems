Typesystems

Status: Open Alpha

Typesystems is a c++ library that aims to provide a way to represent a external typesystem within a c++ program. The library provides a way to;

  1. Identify unknown types
  2. Rewrite type and vlaue information to compatiable types
  3. Store and retrive rewritten variables between typesystems.
